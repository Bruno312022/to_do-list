# Basic REST API Project

This project is a simple REST API built with **Node.js**, **Express**, **Sequelize**, and **MySQL**.  
It was created to practice the fundamentals of backend development, including routing, database modeling, and CRUD operations.  
All API tests were performed using **Thunder Client**.
